### 1. 統計學觀點：大數法則

#### 小樣本問題（之前：MAX_LEN=7, 13-55 個 itemsets）

```
真實 frequent itemsets 很少
→ 每個 itemset 都很「關鍵」
→ 門檻附近的 itemsets 佔比大
→ 小的系統性偏差就會影響 Output Ratio
```

**示例**：

- 真實有 13 個 itemsets
- 估計漏掉 2 個 → Ratio = 2/13 = **15.4%**（很高！）
- 系統抽樣的偏差容易被放大

#### 大樣本優勢（現在：MAX_LEN=None, 129-580 個 itemsets）

```
真實 frequent itemsets 很多
→ 統計特性更穩定
→ 門檻附近的 itemsets 佔比小
→ 系統抽樣的均勻覆蓋優勢顯現
```

**示例**：

- 真實有 580 個 itemsets

- 偏差被更多正確的估計「稀釋」

### 2. 兩個指標的本質差異

#### Support Error Rate（共同 itemsets 的準確度）

```
Error = mean |sup_hat(I) - sup(I)| / sup(I)
        for I ∈ (真實集合 ∩ 估計集合)
```

**特性**：

- 衡量「連續值估計的精確度」
- 只看共同找到的 itemsets
- 受方差影響大 → 系統抽樣能有效降低方差 ✅
- **在任何規模下都能受益於方差降低**

#### Non-common Output Ratio（集合完整度）

```
Ratio = (|假陽性| + |假陰性|) / |真實集合|
      = (估計有但實際沒有 + 實際有但估計沒有) / |真實集合|
```

**特性**：

- 衡量「二元判斷的正確性」（在門檻上方/下方？）
- 關注門檻附近的 itemsets
- 受系統性偏差影響大 ⚠️
- **需要足夠大的樣本空間才能顯示優勢**

### 3. 為什麼 Support Error Rate 持續改善？

系統抽樣的核心優勢是**降低方差**：

```
標準抽樣（有放回）：
- 某些交易可能被重複抽到 5+ 次
- 某些重要交易完全沒被抽到
- 估計值波動大，不穩定

系統抽樣（固定間隔）：
- 整個機率空間被均勻覆蓋
- 每個區域都會被取樣到
- 樣本更具代表性
- 估計值更穩定、更準確
```

**數學原理**：

```
Var(sup_hat_systematic) ≈ (1/k²) * Var(χ/p)
Var(sup_hat_standard)    ≈ (1/k)  * Var(χ/p)

系統抽樣的方差約為標準抽樣的 1/k
```

**新實驗驗證**：

- 在更大的 itemset 空間中，這個優勢更加明顯
- Error Rate 降低幅度：30-60%
- **結論**：在任何規模下，系統抽樣都能有效降低 Error Rate ✅

### 4. 為什麼 Non-common Output Ratio 現在也改善了？

#### 之前的問題：小樣本 + 門檻效應

在小規模實驗（MAX_LEN=7, MIN_SUP=0.02）中：

假設某個 itemset I 的真實 support = 1550，門檻 = 1500：

```
標準抽樣（高方差）：
- Run 1: 估計 = 1480 → 沒找到（假陰性）
- Run 2: 估計 = 1520 → 找到了 ✓
- Run 3: 估計 = 1490 → 沒找到（假陰性）
- 有時對有時錯，但「平均」下來還算合理

系統抽樣（低方差 + 可能有偏差）：
- 方差更小，估計集中
- 如果有系統性偏差 -20，估計會集中在 1490 附近
- → 幾乎「每次」都是假陰性
- → Output Ratio 反而更差 ⚠️
```

**問題**：當 itemset 總數很少時，門檻附近的 itemsets 佔比大，系統性偏差的影響被放大。

#### 現在的改善：大樣本 + 統計平滑

在大規模實驗（MAX_LEN=None, MIN_SUP=0.005）中：

```
更多的 itemsets（129-580 個）
→ 門檻附近的 itemsets 佔比變小
→ 大部分 itemsets 離門檻較遠
→ 系統性偏差的影響被「稀釋」

例如：
- 580 個 itemsets 中，可能只有 20-30 個在門檻附近
- 即使這些受到偏差影響，剩餘 550 個都正確
- → Output Ratio 仍然很好 ✅
```

**關鍵發現**：系統抽樣降低了「隨機誤差」，雖然可能引入小的「系統性偏差」，但在大樣本下，這個偏差的影響被顯著降低！

## 🔬 深入理解：統計學原理

### 1. 方差-偏差權衡（Variance-Bias Tradeoff）

```
MSE = Bias² + Variance

標準抽樣（有放回）：
- Bias ≈ 0（無偏估計）
- Variance 高（隨機性大）
- MSE = 0 + High Variance

系統抽樣（固定間隔）：
- Bias 可能 > 0（可能有小偏差）
- Variance 低（均勻覆蓋）
- MSE = Small Bias² + Low Variance
```

**關鍵發現（新實驗）：**

在大規模實驗中：

```
Variance 降低 ≫ Bias² 增加
→ 系統抽樣的 MSE < 標準抽樣的 MSE
→ 在兩個指標上都表現更好 ✅
```

**為什麼大規模有利？**

```
小規模（13-55 個 itemsets）：
- Bias 的影響相對明顯
- 門檻附近的 itemsets 佔比大
- 小偏差 → 大 Output Ratio 誤差

大規模（129-580 個 itemsets）：
- Bias 的影響被稀釋
- 門檻附近的 itemsets 佔比小
- 同樣的偏差 → 小 Output Ratio 誤差
```

### 2. 門檻效應的規模依賴性

**連續值估計（Support Error Rate）：**

```
小偏差在連續值上：
|估計值 - 真實值| / 真實值 ≈ 1-5%
→ 任何規模下都不明顯
→ 系統抽樣的低方差優勢顯現 ✅
```

**二元判斷（Non-common Output Ratio）：**

```
小規模（55 個 itemsets）：
門檻附近的 itemsets: ~10 個（18%）
小偏差導致錯誤判斷: 3-5 個
Output Ratio: 5/55 = 9% ⚠️

大規模（580 個 itemsets）：
門檻附近的 itemsets: ~30 個（5%）
小偏差導致錯誤判斷: 3-5 個
Output Ratio: 5/580 = 0.9% ✅
```

**結論：規模越大，系統抽樣的優勢越明顯！**
